# Installation
1. Clone repo
2. `yarn install`
3. `cp .env.example .env`
4. Fill the .env file with the configuration.
5. `yarn start`

# Dependencies




1. Material UI
2. APISauce
3. Ramda
4. Redux
5. Redux Persist
6. Redux Saga
7. ImmutableJS
